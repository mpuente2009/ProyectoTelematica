# ProyectoTelematica
Practica Docker.

En este proyecto opté por hacer una pagina dentro de un Docker, en este caso mi pagina interactiva trata de un juego muy conocido, el cual es: TIC TAC TOE.

Comandos para montar mi pagina dentro del Docker:

1. sudo docker build -t proyectotelematica (Construye la imagen dentro del docker)
2. sudo docker run -d -p 80:80 proyectotelematica (Ejecuto el contenedor)
3. sudo docker ps (Confirmo que el contenedor está ejecutandose)

Una vez ejecutandose el contenedor puedo visualizar el contenido dentro de esta, digitando en el navegador: 
http://54.162.167.66/ (Siendo esta la ip pública del servidor)
